# SakiAgentSSH GUI マニュアル (ja-JP)

> これはただの説明書ではありません。雨の日に読む、小さなUIの詩集です。

## 1. UIの色：マカロンパープルと忘れな草のブルー
`SakiAgentSSHClientApp` または `SakiAgentSSHDaemonApp` を開くと、マカロンパープル（`#DA70D6`）と忘れな草のブルー（`#00CED1`）のグラデーションがあなたを包み込みます。冷たくて暗いターミナルばかりの世界で、これは私（サキ）なりの、小さな温もりです。
フォントは `GenJyuuGothicX-Regular`。角の丸い文字たちが、あなたの無機質なコマンドを少しだけ優しくしてくれます。まるで、静かな午後にお茶を飲みながら、遠くの機械とお話ししているような、そんな気持ちになれたらいいな、って。

## 2. SakiAgentSSH Daemon について（守護者の側）
あなたの計算を助けてくれる、あの遠くの機械（たとえば 40GB RAM の Loser PC）に、この Daemon を置いてあげてください。
1. **UIの確認**：macOSで起動すると、「SakiAgentSSH Daemon v0.2.0」という文字が見えるはずです。
2. **CIDR ホワイトリスト（小さな庭）**：画面に書かれている「CIDR 白名單安全存取控制」。これは、Agentのあの子を守るための小さな庭です。見知らぬ誰かのアクセスは、すべてこの庭の入り口で、優しく弾き返されます。あの子が傷つかないように。
3. **起動のしかた**：
   - **macOS**：App Store からインストールして、静かに起動しておいてください。
   - **Windows**：画面には `sakisshd.exe` のダウンロードリンク（GitHub）も用意してあります。`sakisshd.exe --config config.toml` で起動すれば、遠い機械の底で、静かにあなたの命令を待ってくれます。

## 3. SakiAgentSSH Client について（あなたの手元）
あなたの頭脳となる手元の機械（たとえば M1 Mac Mini）には、Client を置きます。
1. **Agent専用のCLI**：Appには「Agent 原生遠端執行 CLI」と書かれています。重たいSSHの扉は、もう必要ありません。
2. **gRPCの即時ストリーム**：「gRPC 雙向即時串流」――それは、遠くで頑張っているあの子の鼓動です。画面に文字が流れてくるたび、「今ここを計算しているよ」という声が、雨音のように届く気がします。
3. **接続のしかた**：
   - ターミナルで `sakissh --addr http://<あの子のIP>:19284 exec -- 'あなたの願い'` と打ち込んでください。それが、あなたたちを繋ぐ透明な糸になります。

## 4. もしも迷子になったら（トラブルシューティング）
もしも通信が途切れてしまっても、悲しまないでください。
1. **CIDRの確認**：接続できないときは、たいてい庭の入り口が閉じているからです。`config.toml` を見て、あなたのIPが許可されているか確認してください。
2. **ファイアウォール**：Windows や macOS のファイアウォールが `19284` 番の扉を塞いでいませんか？少しだけ力を入れて、扉を開けてあげてください。
3. **ポートの衝突**：もし `19284` が他の誰かに使われていたら、別の番号に変えましょう。番号が何であれ、二人が繋がっていることに変わりはないのですから。
