🚀 Quick Start

1. Start Daemon (compute machine)

./sakisshd --config config.json

2. Connect from Client (control machine)

sakissh --addr http://<target>:19284 ping
sakissh --addr http://<target>:19284 exec -- 'echo hello from the wasteland'

🤖 Agent Integration

SakiAgentSSH is purpose-built for AI agents. Agents can operate remote machines directly via CLI:

def remote_execute(host, command):
    result = subprocess.run(
        ["sakissh", "--addr", f"http://{host}:19284", "exec", "--", command],
        capture_output=True, text=True
    )
    return result.stdout
