📦 インストール

macOS — Homebrew Cask

brew tap saki-tw/tools
brew install --cask sakiagentssh-daemon
brew install --cask sakiagentssh-client

Windows — Scoop

scoop bucket add sakistudio https://github.com/Saki-tw/Scoop-SakiStudio
scoop install sakiagentssh-daemon
scoop install sakiagentssh-client

Windows — Winget

winget install SakiStudio.SakiAgentSSH.Daemon
winget install SakiStudio.SakiAgentSSH.Client

🚀 クイックスタート

1. Daemon を起動（ターゲットマシン）

./sakisshd --config config.json

2. Client から接続（コントロールマシン）

sakissh --addr http://<target>:19284 ping
sakissh --addr http://<target>:19284 exec -- 'echo hello'

🤖 Agent 連携

SakiAgentSSH は AI Agent 専用に設計されています。Agent は CLI 経由でリモートマシンを直接操作できます：

def remote_execute(host, command):
    result = subprocess.run(
        ["sakissh", "--addr", f"http://{host}:19284", "exec", "--", command],
        capture_output=True, text=True
    )
    return result.stdout
